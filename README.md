# 💸 RachaAí – Protótipo UX

## 👤 Autor

Isabela, Olavo, Lucas Paulino, Matheus Felipe, Guilherme e Lucas Silva

## 🎯 Persona

Estudantes universitários que dividem apartamento e precisam organizar despesas coletivas de forma simples e sem conflitos.

## 🧠 Heurística de Nielsen

**Visibilidade do Status do Sistema**
O protótipo prioriza mostrar claramente quem deve, quanto deve e o status dos pagamentos, reduzindo incertezas e conflitos.

## 🔄 Fluxo de Tarefa (Dividir Conta de Luz)

1. Acessar o **Dashboard de Grupos**
2. Selecionar o grupo "Casa"
3. Clicar em **Adicionar Gasto**
4. Inserir:

   * Descrição: "Conta de Luz"
   * Valor total
   * Selecionar participantes (3 pessoas)
   * Escolher divisão igual
5. Confirmar o gasto
6. Sistema calcula automaticamente os valores individuais
7. Valores aparecem no **Extrato de Dívidas**

## 🧩 Telas do Protótipo

* Dashboard de Grupos
* Detalhes do Grupo
* Adicionar Gasto
* Scanner de Nota Fiscal
* Extrato de Dívidas
* Confirmação de Pagamento

## 🎯 Objetivo

Facilitar a divisão de despesas coletivas com clareza, rapidez e menos atritos entre usuários.
